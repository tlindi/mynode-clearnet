FQDN for MyNode Apps can be set by creating src file: https_public.conf.src

And then sed with FQDN from file https_domain to https_public.conf 

sudo sh -c '
  domain=$(cat ....clearnet/app_data/https_domain)
  sed "s/\$https_domain/$domain/g" \
    ....clearnet/app_data/https_public.conf.src \
    > /etc/nginx/sites-enabled/https_public.conf
'

server {
    include /etc/nginx/mynode/https_domain.conf;

    listen 6000 ssl;
    server_name <appname>.$https_domain;
...
